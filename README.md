# AI Clinic Management + Smart Diagnosis

A modern, responsive Clinic Management System built for the Mid-Hackathon (Intermediate Level). This project digitizes clinic operations and provides intelligent AI-assisted diagnostics.

## 🚀 Tech Stack (Mid-Hackathon Option 2)
- **Frontend**: HTML5, Vanilla CSS3, JavaScript (ES6+)
- **Backend/Database**: Firebase Authentication & Cloud Firestore
- **Analytics**: Chart.js (Real-time data visualization)
- **Icons**: Lucide Icons

## ✨ Key Features
- **Admin Dashboard**: Real-time staff and patient management, live analytics charts, and revenue tracking.
- **Role-Based Access**: Secure login for Admin, Doctors, Receptionists, and Patients.
- **Dynamic Search**: Live filtering for all user records.
- **Real-Time Snapshots**: Automated UI updates whenever Firestore data changes.

## 🛠 Setup & Installation
1. Clone the repository.
2. Open `index.html` in any modern browser (or use Live Server).
3. Ensure your Firebase configuration in `js/firebase-config.js` is correct.

## 📈 Analytics Info
The charts in the Admin Dashboard are driven by real-time Firestore listeners. Any change to appointments or patient registrations will reflect instantly on the dashboard charts.
